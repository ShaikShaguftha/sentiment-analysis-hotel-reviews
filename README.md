
# Sentiment Analysis on Hotel Reviews

## What This Project Does
Compares two sentiment analysis approaches — VADER and 
Transformer on hotel reviews dataset to analyze 
customer sentiment patterns.

## Key Finding
VADER and Transformers disagreed on a significant portion 
of reviews showing how rule based models read individual 
words while Transformers understand full sentence context.

## Tech Used
- Python
- Pandas
- VADER (NLTK)
- Transformers
